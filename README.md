# Claude Code Free Portuguese

Este projeto pretende disponibilizar uma base em português para estudar, adaptar e utilizar ferramentas open source inspiradas no ecossistema Claude Code.

## Objetivos

- Traduzir documentação.
- Facilitar a instalação.
- Disponibilizar exemplos.
- Criar guias para utilizadores portugueses e brasileiros.

## Estrutura

- docs/ → documentação
- src/ → código
- examples/ → exemplos
